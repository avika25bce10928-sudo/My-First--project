# Simple Dictionary-Based Calculator


##  📌  Overview
This project is a simple command-line calculator built using Python.  
It demonstrates the use of **functions**, **dictionary-based function mapping**, **input handling**, and **basic error validation**.  
Users can perform four basic arithmetic operations: Addition, Subtraction, Multiplication, and Division.


---


##  ✨  Features
- Addition of two numbers  
- Subtraction of two numbers  
- Multiplication of two numbers  
- Division with zero-division error handling  
- Clean and modular code using function mapping  
- Menu-driven user input system


---


##  🛠️  Technologies Used
- **Python 3.x**
